### 专题详情

#### 宋世超 · AI/ML 技术解析与人文思考

覆盖扩散模型原理精注、推测解码算法构造、Python 包设计规范、C/C++ 算法笔记及 Python 爬虫实践，兼有对 AI 与人的关系的技术人文讨论。

#### 字阳 · Linux 内核与 QEMU 虚拟化

系统性记录 Linux 内核从编译、调试（QEMU+GDB+VSCode）、崩溃定位（faddr2line）、Patch 提交，到 VFIO-PCI 显卡直通、XNU 内核 Fuzzing 环境搭建、虚拟设备驱动开发的完整实践链路。

#### 李修羽 · LLM 强化学习前沿

聚焦大语言模型推理与训练前沿：Test-Time Scaling 与免训练 RL、Agentic RL 中 Reasoning 与 Tool-use 的解耦，以及 RLHF 泛化与收敛的理论可证明界。

#### 熊洋铭 · LeetCode 每日一题

以二进制运算为主线，涵盖二叉搜索树平衡、平衡子数组/子串、香槟塔模拟等题型，持续记录每日算法解题过程与思路。

#### 王宇航 · verl 框架代码拆解

逐层解析 verl（分布式 RL 训练框架）的源码结构，包括训练入口、基础脚手架、`DataParallelPPOActor`、`Rollout`、`ActorRolloutRefWorker` 五大核心模块。

#### 罗翌宁 · LeetCode 题解

双指针解法解析 LeetCode 11「盛最多水的容器」。

#### 周子涵 · 深度学习基础知识

#### 胡博玮 · AI Agent 辅助项目方法论

探讨在 AI Agent 辅助下高效完成工程项目的方法论。
